# sofit-viewer

### Сервер (порт 3001)
```
cd ./server
npm i
node index.js
```
Для демонстрационных целей в папке `/server/storage` лежат 30 трейсов. Можно в неё распаковать весь архив с трейсами — фронт умееет пагинировать по скроллу с шагом в 20 айтемов.  

### Клиент (порт 3000)
```
cd ./client
npm i
npm start
```
